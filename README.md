Sales performance analysis using Python, Pandas, and Matplotlib
# Python Sales Performance Analysis

## Project Overview
This project analyzes retail sales transaction data using Python to uncover revenue trends, identify top-performing products, and evaluate regional performance. The goal was to transform raw transactional data into actionable business insights using data analysis and visualization techniques.

---

## Tools & Technologies
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

---

## Data Preparation
- Created structured dataset of sales transactions
- Calculated Revenue column (Quantity × Unit Price)
- Converted date fields for time-based analysis
- Aggregated data using groupby() operations

---

## Business Questions Answered
1. What is the total revenue generated?
2. Which product generates the highest revenue?
3. Which region performs best?
4. How does revenue trend over time?

---

## Key Insights
- Total Revenue: **$7,055**
- Top Product: **Laptop**
- Top Region: **North**
- January recorded the highest monthly revenue
- Revenue fluctuates across months indicating seasonal variation

---

## Project Structure
- `sales_analysis.ipynb` → Jupyter notebook with full analysis
- `sales_analysis.html` → Exported report version
- `sales_data.csv` → Dataset used
- Screenshots → Visual proof of results
